Q1.Problem:

You are given a string (STR) of length N, consisting of only the lower case English alphabet.

Your task is to remove all the duplicate occurances of characters in the string.

input:

abcadeecfb

output:

abcdef

    Ans. function remove1(str) {
        let result = '';
        const seenChars = new Set();
        for (let char of str) {
        if (!seenChars.has(char)) {
        result += char;
        seenChars.add(char);
        }
        }
        return result;
        }
        const inpu = 'abcadeecfb';
        const outp = remove1(inpu);
        console.log(outp);

Output: abcdef

Explain: for Removing duplicate element in the list first we assign empty string to store value and after that we check the duplicate element if duplicate elemnt is found we remove it and If the character is not a duplicate, add it to the result string and add it to the set. return the upadted output.

Q.2 Problem:

You are given a string(STR) of length N, you have to print the count of all alphabet.(using map)

input : abcadeecfb

Ans: the code is Here

      function count(str) {
      const charCountMap = new Map();
      for (let char of str) {
      if (charCountMap.has(char)) {
      charCountMap.set(char, charCountMap.get(char) + 1);
       } else {
        charCountMap.set(char, 1);
       }
      }
      for (let [char, count] of charCountMap) {
      console.log(`${char}=${count}`);
        }
      }
      const input = 'abcadeecfb';
      count(input);
